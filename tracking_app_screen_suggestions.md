# Tracking App - Screen Suggestions

## 1. Authentication Screens
- **Splash Screen** - App launch with branding
- **Login Screen** - Email/password authentication
- **Sign Up Screen** - New account creation
- **Password Reset Screen** - Account recovery

## 2. Core Tracking Screens
- **Dashboard/Home Screen** - Overview of active challenge, progress, quick stats
- **Challenge Selection Screen** - Browse/select from available challenges
- **Challenge Setup Screen** - Define goal, target value, duration, category
- **Log Entry Screen** - Record performance data with date/time
- **Challenge Details Screen** - View full challenge info, progress bar, entry history, charts

## 3. Progress & Analytics Screens
- **Progress Details Screen** - Detailed metrics (percentage, days remaining, current streak)
- **Analytics Screen** - Charts showing progress over time (line chart, bar chart)
- **Achievements/Milestones Screen** - Display earned badges and unlocked milestones

## 4. User Account Screens
- **Profile Screen** - Display user info and account statistics
- **Edit Profile Screen** - Update user information
- **Edit Challenge Screen** - Modify existing challenge parameters

## 5. Settings Screens
- **Settings Screen** - Notifications, theme, privacy, data management
- **Notification Settings Screen** - Configure reminders and alerts

## 6. Navigation
- **Bottom Navigation Bar** - Home, Log Entry, Challenges, Analytics, Profile
- **Floating Action Button (FAB)** - Quick access to log new entry
- **Drawer Menu** - Settings, Help, About (optional)

## 7. Optional Screens (Phase 2)
- **Team/Group Screen** - View team members and team progress (if collaborative)
- **Leaderboard Screen** - Competitive rankings (if social feature enabled)
- **Help/FAQ Screen** - User guidance and support

## Summary of Minimum Screens Required
**Essential (P0):** Dashboard, Log Entry, Challenge Setup, Challenge Details, Profile, Settings (6 screens minimum for MVP)

**Recommended (P1):** Analytics, Achievements, Edit Challenge (adds 3 more = 9 total)

**Nice to Have (P2):** Team features, Leaderboard, Help (adds 3 more = 12 total)
